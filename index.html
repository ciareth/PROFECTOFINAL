PROYECTO 1

ESTE CODIGO PARA HACER UN FORMULARIO LO EXTRAIMOS DEL LIBRO DE DISEÑO WEB, SOLAMEMENTE TUVIMOS QUE SEGUIR LAS INSTRUUCCIONES DEL LIBRO, Y SI COPIABAMAMOS CORRECTAMENTE EL CODIGO DEBIA APARECERNOS TAL Y COMO ESTA EN EL LIBRO.

PROYECTO 2

FORMULARIO, EN ESTA CASO EL LIBRO NOS DIO EL CODIGO HTML, NOSOTROS DEBIAMOS DARLE FORMA Y SENTIDO CON NUESTRA CREATIVIDA UTILIZANDO CSS.

PROYECTO 3

UN FORMULARIO, EL LIBRO NOS BRINDABA EL CODIGO PERO EN PHP, CON AYUDA DE LA IA PUDE CONVERTIR ESE CODIGO EN HTML, Y TUVE QUE DARLE FORMA UTILIZANDO CSS, TENIA UN MODELO AL CUAL SEGUIR Y TRATE DE HACER ESTE LO MAS PARECIDO AL QUE MOSTRABA EL LIBRO.

PROYECTO 4

EN ESTE PROYECTO CUANDO DAMIS CLIC EN LA IMAGEN AUTOMATICAMENTE HACE ZOOM. NUEVAMENTE EL LIBRO NOS BRINDABA UN CODIGO PERO EN PHP, PEDI AYUDA A CHAT GPT Y ME DIO EL HTML, Y TAMBIEN UN CODIGO DE JAVASCRIPT, AL CUAL YO LE TUVE QUE ANEXAR UN CSS PARA QUE SE EJECUTARA EL ZOOM DE LAS IMAGENES.
